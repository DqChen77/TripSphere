二、靶场两条 AI 链路

链路 A — 订单链（A2A + 多跳 gRPC，5 跳，Python→Java）
```
Browser → Next.js → chat-service (ADK) → [A2A] → order-assistant (ADK)
  → [gRPC] → order-service → [outbound gRPC] → product-service
                            → [outbound gRPC] → inventory-service
```

链路 B — 行程规划链（LangGraph AG-UI，双 gRPC，Python→Java）
```
Browser → Next.js → itinerary-planner (LangGraph AG-UI)
  → [gRPC] → attraction-service (MongoDB)
  → [gRPC] → hotel-service (MongoDB)
  → [LLM]  → Higress → 外部 LLM
  → [gRPC] → itinerary-service (保存)
```
